# The Quest Board

The Quest Board is an AI-powered instructional planning tool for K–12 teachers. It helps teachers turn source content into differentiated, classroom-ready learning experiences.

## What this project does
- Generates Choice Boards
- Generates WebQuests
- Generates Socratic Seminars
- Supports PDF, DOCX, and TXT upload
- Extracts source text
- Supports DOK level selection
- Includes export preview
- Includes student worksheet preview
- Includes a visual mastery tracker

## Current goal
The current goal is to move from prototype to a real production website.

## MVP focus
Phase 1 priority:
- Choice Boards first

Later phases:
- WebQuests
- Socratic Seminars
- Google Drive integration
- Google Classroom integration
- Teacher accounts and saved libraries
- Paid subscriptions

## Product vision
The Quest Board helps teachers save time, maintain rigor, differentiate instruction, and create engaging learning journeys from one source text.

## Status
This is currently a prototype/demo build, not yet the full production version.
